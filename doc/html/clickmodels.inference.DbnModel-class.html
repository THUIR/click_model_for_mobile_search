get_model_relevances(self, session, intent=False)

source code 

Returns estimated relevance of each document in a given session based on a trained click model.

You can make use of the fact that model trains different relevances for different intents by specifying `intent` argument. If it is set to False, simple web relevance is returned, if it is to True, then vertical relevance is returned, i.e., how relevant each document is to a vertical intent.
