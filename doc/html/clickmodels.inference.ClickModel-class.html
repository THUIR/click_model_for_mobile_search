Method Details [hide private]

test(self, sessions, reportPositionPerplexity=True)

source code 

Evaluates the prediciton power of the click model for a given sessions. Returns the log-likelihood, perplexity, position perplexity (perplexity for each rank a.k.a. position in a SERP) and separate perplexity values for clicks and non-clicks (skips).

_get_click_probs(self, s, possible_intents)

source code 

Returns click probabilities list for a given list of s.clicks. For each intent $i$ and each rank $k$ we have: click_probs[i][k-1] = P(C_1, ..., C_k | I=i)

get_loglikelihood(self, sessions)

source code 

Returns the average log-likelihood of the current model for given sessions. This is a lightweight version of the self.test() method.

get_log_click_probs(self, session)

source code 

Returns an average log-likelihood for a given session, i.e. log-likelihood of all the click events, divided by the number of documents in the session.

predict_click_probs(self, session, intent=False)

source code 

Predicts click probabilities for a given session. Does not use session.clicks. This is a vector of P(C_k = 1 | E_k = 1) for different ranks $k$.

predict_stop_probs(self, session, intent=False)

source code 

Predicts stop probabilities (after click) for each document in a session. This is often referred to as satisfaction probability. This is a vector of P(S_k = 1 | C_k = 1) for different ranks $k$.
